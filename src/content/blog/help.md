# How to use this blog

<br>

## #COMMAND MODE

```ts
Press `:` to enter command mode. After typing your command, press `Enter` to execute it.
```

<br>

| command | action              |
| ------- | ------------------- |
| q       | Return to home page |
| help    | Go to the help page |
| esc     | Exit command mode   |
